# Vibe Coding Workshop — Starter Files

This project contains the starter files for the Vibe Coding workshop. You'll paste AI-generated code from Claude (or ChatGPT/Gemini) into these files to build your portfolio website.

## What's Inside

- **index.html** — The main page (paste HTML here)
- **style.css** — How it looks (paste CSS here)
- **script.js** — How it works (paste JavaScript here)

## Getting Started

### Option A: Open in Phoenix Code (easiest)

1. Go to **phcode.dev** and click **"Start Coding"**
2. In the welcome dialog, click **GitHub Import**
3. Paste: `curtrode/vibe-coding-starter`
4. Click the **Live Preview** button (lightning bolt icon) to see your site

### Option B: Download to your computer

#### Step 1 — Download

1. On this page, click the green **Code** button near the top
2. Click **Download ZIP**
3. A file called `vibe-coding-starter-main.zip` will download to your computer

#### Step 2 — Unzip the file

**On Mac:**
- Find the ZIP file in your **Downloads** folder
- Double-click it — a new folder called `vibe-coding-starter-main` will appear right next to it
- That's it! Open the folder to see your files

**On Windows:**
- Find the ZIP file in your **Downloads** folder
- Right-click the file and choose **Extract All...**
- Click **Extract** in the window that pops up
- A new folder called `vibe-coding-starter-main` will open with your files inside

#### Step 3 — Open in Phoenix Code

1. Go to **phcode.dev** and click **"Start Coding"**
2. In the welcome dialog, click **Open Folder**
3. Navigate to the `vibe-coding-starter-main` folder you just unzipped and select it
4. Click the **Live Preview** button (lightning bolt icon) to see your site

> **Note:** Open Folder works in Chrome, Edge, and Opera. If you're using Firefox or Safari, use the GitHub Import option instead.

## During the Workshop

You'll use Claude (claude.ai) to generate code, then paste it into the matching file:

| Claude gives you... | Paste into... |
|---------------------|---------------|
| HTML code | **index.html** |
| CSS code | **style.css** |
| JavaScript code | **script.js** |

The Live Preview in Phoenix Code updates automatically when you save (Ctrl+S on Windows, Cmd+S on Mac).
